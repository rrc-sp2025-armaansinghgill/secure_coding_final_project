# Code Security Audit - Community Support Tracker

## Overview
This project is a simple web application used to track community donations.  
It is used for performing a **security audit** as part of a course assignment.

## Purpose
- Identify security vulnerabilities in the code
- Map issues to **OWASP Top 10 (2025)**
- Perform peer code review
- Use automated security tools

## Technologies
- HTML, CSS, JavaScript
- Node.js (for testing)
- GitHub

## Security Focus
The project includes intentionally vulnerable code to demonstrate:
- Input validation issues
- Unsafe DOM handling
- Insecure data storage
- Exposure of sensitive information

## How to Run
1. Clone the repo  
2. Open `index.html` in a browser
